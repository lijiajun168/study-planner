# 留学方案推荐网页

这是一个静态网页应用，打开 `index.html` 或用本地服务器访问即可使用。

## 当前功能

- 首页选择目标地区：英国、中国香港、澳洲、爱尔兰、新加坡
- 第二页填写昵称、本科学校、本科专业、百分制均分
- 提交后生成 6 个学校和专业方案
- 推荐逻辑结合本科院校分档、均分阈值、专业相关性和学校排名顺序
- 每个方案包含学校、专业、规则线、推荐理由和官网专业链接

## 数据维护

主要数据在 `data.js`：

- `tiers`：中国本科院校分档
- `thresholds`：不同地区对不同分档的基础均分要求
- `programs`：推荐项目库、项目链接和分档规则线

公开部署版说明：

- GitHub Pages 版本使用官网链接和公开版通用初筛规则
- 牛津、剑桥、帝国理工等高排名院校已加入英国候选池，采用官网项目链接和公开版通用初筛分数
- 英国推荐会区分“名单限制项目”和“接受范围较宽项目”；本科院校无法匹配到内置名单时，不推荐名单限制项目
- 本科背景细分为 985、211、一本双非/公办本科、三本民办/独立学院；自动识别不确定时按公办本科处理，也可在表单中手动指定
- 香港、澳洲、爱尔兰、新加坡目前仍使用通用分档规则，后续放入对应地区文档后可继续拆成独立规则文件

后续如有内部项目文档，可把文档中的分档名单和项目要求补入独立规则文件，不需要改页面结构。
